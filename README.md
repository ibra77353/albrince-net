# البرنس نت v1.0

الإصدار الأول الرسمي من تطبيق البرنس نت.

## المميزات

- مشاهدة الاستراحة.
- مشاهدة البث المباشر.
- واجهة بسيطة وسريعة.
- إضافة زر تحديث صفحة الدخول.
- إضافة شريط تحميل أثناء فتح الصفحات.
- إضافة مشاركة التطبيق.
- إضافة صفحة "حول التطبيق".
- التحقق من وجود تحديثات جديدة.
- تحسين معالجة أخطاء الاتصال.
- عرض رسالة واضحة عند عدم توفر الإنترنت.
- تحسينات عامة في الأداء والاستقرار.

## متطلبات التشغيل

- Android 5.0 (API 21) أو أحدث.
